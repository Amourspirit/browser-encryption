Security Level

Encryption:

  1. Choose Algorithm (AES, 3DES ..)
  2. Enter or generate your secret key (AKA password)
  3. Copy/paste or write your text you want to encrypt
  4. Click Encrypt
  5. Click on Download to save your file
  6. Click Gen URL to create a link that contins the encryptd text ( without key ) that can be shared

Decryption:

  1. Choose Algorithm (AES, 3DES ..)
  2. Enter your secret key you used for encryption (AKA password)
  3. Copy/paste or write your encrypted text you want to decrypt
  4. Click decrypt
  5. Click on Download to save your file

URL:

  • After encryption optinally generate a url
  • Url can be shared publicly or privatly, its up to you
  • Copy url and share the link
  • Url does not contain the key unless you specifically Include Key check-box
  • Url link will populate this page when clicked. Only Key need be entered (if key was not specifically included) and then click Decrypt
